## Project Summary

Baseerah AI is a smart city monitoring platform designed to analyze and visualize operational indicators for urban environments.

The project includes:

- Web Dashboard
- Backend API
- PostgreSQL Database
- Real-Time Monitoring
- AI Assistant
- Mobile Application

The current prototype focuses on **Abha, Saudi Arabia** as the main smart city environment.

## Known Limitations

The current version is mainly designed for local development.

Limitations:

- Backend currently runs locally.
- PostgreSQL database currently runs locally.
- Mobile app requires the correct API address when running locally.
- Cloud deployment is not completed yet.
- IoT sensor integration is not implemented yet.
- AI analysis is currently based on available city indicators.
